# 業務管理システム

Laravel 12を使用した業務管理システムのポートフォリオプロジェクトです。

## 機能

- **認証システム**: ユーザー登録・ログイン機能
- **プロジェクト管理**: プロジェクトの作成・編集・削除、メンバー管理
- **タスク管理**: タスクの作成・編集・削除、ステータス管理、優先度設定
- **工数管理**: 工数の記録・管理、プロジェクト・タスクごとの集計
- **ダッシュボード**: 統計情報の表示、最近のプロジェクト・タスクの一覧

## 技術スタック

- **フレームワーク**: Laravel 12
- **データベース**: SQLite（開発環境）/ MySQL（本番環境）
- **フロントエンド**: Blade テンプレート + Tailwind CSS
- **認証**: Laravel標準認証

## データベース構造

### テーブル

- `users`: ユーザー情報
- `projects`: プロジェクト情報
- `tasks`: タスク情報
- `time_entries`: 工数記録
- `project_user`: プロジェクトとユーザーの多対多リレーション

### リレーション

- プロジェクトは複数のユーザー（メンバー）を持つ
- プロジェクトは複数のタスクを持つ
- タスクは1人のユーザーに割り当て可能
- 工数記録はプロジェクトとタスクに関連付けられる

## 主な機能の説明

### プロジェクト管理

- プロジェクトの作成・編集・削除
- プロジェクトステータス管理（計画中、進行中、保留、完了、キャンセル）
- メンバーの追加・削除
- プロジェクトごとのタスク・工数管理

### タスク管理

- タスクの作成・編集・削除
- タスクステータス管理（未着手、進行中、レビュー中、完了）
- 優先度設定（低、中、高、緊急）
- 担当者の割り当て
- 見積時間の設定

### 工数管理

- プロジェクト・タスクごとの工数記録
- 日付・時間・説明の記録
- 工数の集計表示

## ポートフォリオとしての特徴

このプロジェクトは、以下の技術要素を含んでいます：

1. **認証・認可**: ユーザー認証、ロール管理
2. **CRUD操作**: プロジェクト、タスク、工数の完全なCRUD
3. **リレーション**: Eloquent ORMを使用した複雑なリレーション
4. **バリデーション**: フォームバリデーション
5. **UI/UX**: Tailwind CSSを使用したモダンなUI
6. **データベース設計**: 適切な正規化とリレーション設計
